# 智慧建築六大指標與弱電關聯摘要

## 基礎設施

### 可能相關系統
- 資訊網路系統
- 通訊系統
- 中央監控 / 建築管理系統
- 弱電基礎佈線
- 公眾通信涵蓋
- 系統整合平台
- 建築靜態 / 動態數據庫
- 資訊安全與備援

### HTML 呈現方式
- 可能對應指標：基礎設施
- 可能需要佐證：網路拓樸圖、系統架構圖、設備表、通訊介面說明、配線架構圖、測試報告、資安機制說明
- 需人工確認：正式採用項目、佐證格式、審查顧問要求、系統邊界、資料庫與備援範圍

### 弱電設計影響
- 影響垂直主幹、水平佈線、光纖、銅纜與無線網路設計。
- 影響 MDF、IDF、弱電機房、網路設備、通訊涵蓋與配線標示。
- 影響 BMS、CCTV、門禁、停車、對講、消防、感測器等系統資料傳輸基礎。
- 影響網路備援、系統備援、資料庫備援與資訊安全設計。

### 可能需要佐證
- 配線規劃昇位圖
- 配線規劃平面圖
- 竣工架構圖
- 佈線檢測報告
- 無線訊號測試報告
- 資安架構與功能說明
- 備援架構與功能說明

### 需人工確認事項
- 弱電系統是否納入智慧建築申請範圍。
- 公眾通信、無線網路或物聯網服務是自建或租用。
- 資料庫欄位、資料格式、資料交換與保存責任。
- 資安設備、權限、加密、雙因子、軟體更新是否有明確規格。

## 維運管理

### 可能相關系統
- 中央監控 / 建築管理系統
- 維運管理平台
- 行動管理 APP
- I/O 點表
- BIM / 3D 視覺化整合
- 營運數據分析
- 智慧工地安全、人員、資材管理

### HTML 呈現方式
- 可能對應指標：維運管理
- 可能需要佐證：平台架構圖、整合協定、I/O 點表、操作畫面、維運計畫、行動裝置畫面、營運數據分析說明
- 需人工確認：平台功能、整合子系統、運轉紀錄、異常通知、維運責任、是否需要納入施工階段管理

### 弱電設計影響
- 影響 BMS/IBMS 平台功能、子系統整合協定與圖控畫面。
- 影響中央監控點表、警報通知、趨勢紀錄、報表與遠端控制。
- 影響 APP 管理、訪客、郵務、報修、權限控制等服務是否納入。
- 影響維運文件、五年修繕計畫、危機處理與緊急應變資料。

### 可能需要佐證
- 智慧化系統規劃設計圖說
- 智慧化功能 I/O 點數表
- 跨系統互動流程圖
- 平台架構圖與整合協定
- 操作畫面、影片或使用說明
- 維運管理計畫
- 營運數據分析與預測規劃

### 需人工確認事項
- BMS/IBMS 是否只監視或包含控制。
- 子系統介接協定、責任界面與測試方式。
- 異常預警、主動通報、影像跳圖、趨勢圖與報表是否為必要功能。
- BIM/3D 整合是否屬本案需求。

## 安全防災

### 可能相關系統
- 火警系統
- 監視系統
- 門禁系統
- 對講系統
- 停車管理系統
- 防盜保全系統
- 緊急求救系統
- 廣播 / 聲光警報
- 多媒體顯示系統
- 中央監控 / 建築管理系統

### HTML 呈現方式
- 可能對應指標：安全防災
- 可能需要佐證：設備配置圖、系統架構圖、I/O 點表、連動邏輯圖、系統畫面、測試紀錄、竣工圖
- 需人工確認：消防連動、門禁解除、影像連動、緊急求救位置、停車空間、住宿類特殊要求

### 弱電設計影響
- 影響火警、門禁、電梯、空調、送排風、監視、緊急求救、防盜保全等系統連動。
- 影響 CCTV 點位、門禁點位、對講點位、求救裝置與告警顯示裝置。
- 影響災害顯示、聲音告警、即時影像連動跳圖與 BMS 記錄。
- 影響安全系統的操作畫面、測試影片與佐證照片。

### 可能需要佐證
- 消防設備配置圖
- 門禁系統配置圖
- 監視系統配置圖
- 緊急求救系統配置圖
- 停車管理系統配置圖
- 系統連動邏輯圖
- 智慧化功能 I/O 點數表
- 系統畫面與連動測試紀錄

### 需人工確認事項
- 主要逃生動線、重要空間、公共空間與停車空間範圍。
- 火警時門禁解除、電梯迫降、空調與送排風連動是否與消防設計一致。
- 影像連動範圍是否涉及隱私或法規限制。
- 緊急求救是否需整合至戶內對講或行動通知。

## 節能管理

### 可能相關系統
- 能源管理系統
- 中央監控 / 建築管理系統
- 空調監控
- 電力監控
- 照明控制
- 插座控制
- 給排水監控
- 停車場排風監控
- 儲能 / 再生能源監控

### HTML 呈現方式
- 可能對應指標：節能管理
- 可能需要佐證：能源管理系統架構圖、電力單線圖、控制圖、I/O 點表、圖控畫面、測試計畫、測試報告
- 需人工確認：採用率、節能效益、控制或監視、分項用電、需量控制、儲能或創能介面

### 弱電設計影響
- 影響 EMS/BMS 架構、能源數據擷取、圖控畫面與紀錄。
- 影響空調、照明、插座、給排水、停車排風等控制點與監視點。
- 影響電表、水表、感測器、控制器、通訊介面與資料格式。
- 影響節能驗證、效益模擬、測試紀錄與送審資料。

### 可能需要佐證
- 電力單線圖
- 能源管理系統架構圖
- 智慧化功能 I/O 點數表
- 控制系統驗證報告
- 智慧化節能控制系統畫面
- 水平衡圖、給水昇位圖、排水昇位圖
- 用電卸載、儲能、創能控制測試報告

### 需人工確認事項
- 用電、用水、空調、照明、動力是否納入監測。
- 智慧化控制是排程、感測連動、需量卸載或 AI 預測。
- 採用率、節能積分、運轉偏差率、能源績效指標不能由 GPT 判斷。
- 是否需要第三方測試、顧問報告或正式送審格式。

## 健康舒適

### 可能相關系統
- 室內空氣品質監測
- 環境感測器
- 中央監控 / 建築管理系統
- 空調連動
- 照明連動
- 資訊顯示 / APP
- 非接觸式服務
- 健康生活服務平台

### HTML 呈現方式
- 可能對應指標：健康舒適
- 可能需要佐證：感測器配置圖、I/O 點表、系統說明、設備型錄、資訊顯示畫面、連動圖、測試照片或影片
- 需人工確認：感測器位置、數量、顯示方式、異常提醒、控制連動、住宿類公共空間範圍

### 弱電設計影響
- 影響溫濕度、二氧化碳、PM2.5、TVOC、照度、水質、雨量等感測規劃。
- 影響環境資訊顯示、APP、BMS 圖控與異常提醒。
- 影響空調、外氣、照明或其他設備的主動連動。
- 影響非接觸、密度管制、健康服務平台等低接觸智慧化設施。

### 可能需要佐證
- 環境感測設施設置平面圖
- 智慧化功能 I/O 點數表
- 感測設施規範與系統說明
- 設備型錄
- 系統畫面照片
- 主動連動圖
- 現場測試照片或影片

### 需人工確認事項
- 哪些空間需設置環境感測器。
- 感測器數量與位置不得由 GPT 自行決定。
- 顯示、紀錄、查詢、分析與異常提醒是否全部必要。
- 空調或照明連動是否需納入控制權限與測試。

## 智慧創新

### 可能相關系統
- 智慧管理雲平台
- 資料格式標準
- 智慧建材
- AI / 預測演算法
- 中央監控 / 建築管理系統
- 能源管理系統
- 資訊網路系統

### HTML 呈現方式
- 可能對應指標：智慧創新
- 可能需要佐證：創新服務設計說明、裝置圖說、系統架構、情境控制說明、標準驗測報告、雲平台功能規格、操作介面、資安規範
- 需人工確認：創新程度、第三方認證、資料標準版本、雲平台責任界面、是否屬正式申請項目

### 弱電設計影響
- 影響資料交換格式、標準協定、雲平台資料匯入與資安規範。
- 影響系統整合、AI 分析、營運數據、能源數據與設備妥善率應用。
- 影響智慧建材若需通訊、感測、致動或系統整合的介面。

### 可能需要佐證
- 創新加值服務設計說明
- 系統及情境控制說明
- 智慧建材名稱、規格、功能與第三方認證
- 標準資料格式驗測報告
- 智慧管理雲平台系統架構與功能規格
- 操作介面、維運方式、資安規範

### 需人工確認事項
- 智慧創新是否納入本案正式申請。
- 創新程度需由評定或顧問確認。
- 資料格式標準、驗測報告與版本需確認。
- 雲平台是否為政府平台、私有平台或專案自建平台。
